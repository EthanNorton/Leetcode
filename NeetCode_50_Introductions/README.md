# NeetCode 50 - Introduction Problems

This folder contains the introductory problems from each category of the NeetCode 50 list. These are designed to be solved slowly and methodically to build a strong foundation.

## Problem Categories

### 1. Arrays & Hashing
- **Contains Duplicate** - Check if array contains duplicates
- **Valid Anagram** - Check if two strings are anagrams
- **Two Sum** - Find two numbers that add up to target

### 2. Two Pointers
- **Valid Palindrome** - Check if string is palindrome
- **Two Sum II** - Two sum on sorted array

### 3. Sliding Window
- **Best Time to Buy and Sell Stock** - Find max profit from stock prices

### 4. Stack
- **Valid Parentheses** - Check if parentheses are valid

### 5. Binary Search
- **Binary Search** - Classic binary search implementation

### 6. Linked List
- **Reverse Linked List** - Reverse a linked list

### 7. Trees
- **Invert Binary Tree** - Invert a binary tree

### 8. Tries
- **Implement Trie (Prefix Tree)** - Implement a trie data structure

### 9. Backtracking
- **Combination Sum** - Find all combinations that sum to target

### 10. Graphs
- **Number of Islands** - Count islands in a grid

### 11. 1-D Dynamic Programming
- **Climbing Stairs** - Count ways to climb stairs

### 12. 2-D Dynamic Programming
- **Unique Paths** - Count unique paths in grid

### 13. Greedy
- **Maximum Subarray** - Find maximum sum subarray

### 14. Intervals
- **Insert Interval** - Insert and merge intervals

### 15. Math & Geometry
- **Rotate Image** - Rotate 2D matrix 90 degrees

### 16. Bit Manipulation
- **Number of 1 Bits** - Count set bits in number

## How to Use This Folder

1. Work through problems in order, one category at a time
2. Read the problem description in each folder
3. Try solving it yourself first
4. Review the solution template if stuck
5. Add your own notes and insights
