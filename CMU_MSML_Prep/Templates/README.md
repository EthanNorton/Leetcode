## üìÅ How to Use These Templates

### Step 1: Start with Templates
1. Open a template file (e.g., `exercise_1_normalize_template.py`)
2. Read the instructions and requirements
3. Fill in the `TODO` sections
4. Implement the functions from scratch

### Step 2: Test Your Implementation
1. Uncomment the test functions
2. Run: `python exercise_X_template.py`
3. Debug until all tests pass

### Step 3: Compare with Solutions
1. Once you've implemented and tested, look at:
   - `../00_Skill_Exercises/` folder
   - **Side-by-side guides** (e.g., `02_NumPy/SIDE_BY_SIDE_COMPARISON.md`)
   - Compare your approach with the solution
   - Learn from differences
