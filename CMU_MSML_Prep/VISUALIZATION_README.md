# ML Concepts Visualization Generator

**Purpose:** Generate visual diagrams and plots to help understand ML concepts visually.

---

## What It Does

Creates 10 different visualizations:
1. **ReLU Function** - Shows the activation curve
2. **Sigmoid Function** - Shows the S-shaped curve
3. **Activation Comparison** - ReLU vs Sigmoid side-by-side
4. **Gradient Descent Convergence** - Loss decreasing over iterations
5. **Softmax Example** - Scores to probabilities transformation
6. **Feature Scaling Comparison** - Before/after scaling
7. **Matrix Multiplication Visual** - Step-by-step diagram
8. **Reshape Visual** - Matrix reshaping diagram
9. **One-Hot Encoding Visual** - Categorical to binary vectors
10. **Gradient Descent 3D** - Loss surface with descent path

---

## How to Use

### Step 1: Install Requirements
```bash
pip install matplotlib numpy
```

### Step 2: Run the Generator
```bash
python CMU_MSML_Prep/VISUALIZATION_GENERATOR.py
```

### Step 3: View Images
All images will be saved in `CMU_MSML_Prep/visualizations/` folder.
