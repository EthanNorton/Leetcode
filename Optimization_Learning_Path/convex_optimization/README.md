## ðŸ“š Lecture Sequence & Challenges

### Week 1: Foundations
**Lecture 1**: Introduction to Convex Optimization
- **Video**: [Convex Optimization - Introduction](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Implement basic convex function checker
- **Key Concepts**: Convex vs non-convex problems, applications

**Lecture 2**: Convex Sets
- **Video**: [Convex Sets](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Visualize and classify convex sets in 2D/3D
- **Key Concepts**: Affine sets, convex hulls, separation theorems

**Lecture 3**: Convex Functions
- **Video**: [Convex Functions](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Implement convexity tests for various functions
- **Key Concepts**: First-order and second-order conditions

### Week 2: Optimization Problems
**Lecture 4**: Convex Optimization Problems
- **Video**: [Convex Optimization Problems](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Formulate real-world problems as convex optimization
- **Key Concepts**: Linear programming, quadratic programming, SOCP

**Lecture 5**: Duality Theory
- **Video**: [Duality Theory](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Implement dual problem formulation
- **Key Concepts**: Lagrange duality, weak/strong duality, KKT conditions

### Week 3: Algorithms
**Lecture 6**: Gradient Methods
- **Video**: [Gradient Methods](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Implement gradient descent with line search
- **Key Concepts**: Convergence analysis, step size selection

**Lecture 7**: Interior Point Methods
- **Video**: [Interior Point Methods](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Implement barrier method for linear programming
- **Key Concepts**: Central path, barrier functions, complexity

### Week 4: Applications
**Lecture 8**: Machine Learning Applications
- **Video**: [ML Applications](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Implement SVM using convex optimization
- **Key Concepts**: Regularization, support vector machines

**Lecture 9**: Signal Processing
- **Video**: [Signal Processing](https://www.youtube.com/watch?v=McLq1hEq3UY)
- **Challenge**: Implement compressed sensing optimization
- **Key Concepts**: Sparse recovery, compressed sensing
